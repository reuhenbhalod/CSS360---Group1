# GoodEats

A multi-source dining dashboard for Bothell, WA.

## 1. Pull the project

Make sure you have Git installed, then run:

git clone https://github.com/reuhenbhalod/CSS360---Group1.git
cd CSS360---Group1

## 2. Install requirements

You'll need Node.js installed (developed on v24.15.0).
Download it at https://nodejs.org if you don't have it.

Then install project dependencies:

npm install --legacy-peer-deps

## 3. Run the test suite

npm test

You should see output like:
src/test/utils.test.js (36 tests)
src/test/components.test.jsx (27 tests)
